GrowPier

Desenvolvimento de sistema web para gestão de Hackthons e Maratonas:

  • Cadastro de participantes com validação de e-mail
  • Bot para o Discord com validação de cadastro
  • Formação de equipes com requisitos específicos
  • Agendamento de mentorias
  • Comunicação entre mentor e mentorado com um chat interno
  • Submissão de projetos com validação dos arquivos
  • Cadastro de jurados
  • Avaliação dos projetos
  • Inclusão de comentário a equipe
  • Painel de acesso à organização
  • Diferente níveis de acesso divididos por perfil
Atividades realizadas utilizando metodologia ágil SCRUM e MTV como padrão de projeto

Lizard Solution

Desenvolvimento de sistema web para recrutamento e seleção com as seguintes funcionalidades:

  • Batida de ponto
  • Prospecção de leeds
  • Agenda
  • Envio automatizado de propostas
  • Cadastro de vagas
  • Gerenciamento de vagas por recrutador
  • Cadastro de candidatos
  • Candidatura via plataforma
  • Aprovação de candidatos via e-mail pelo gestor
  • Diferente níveis de acesso divididos por perfil
Atividades realizadas utilizando metodologia ágil SCRUM e MTV como padrão de projeto
